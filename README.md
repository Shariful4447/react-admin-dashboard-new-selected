# React Admin Dashboard 


## Technology used
1. React
2. Material UI 

### This application consists of 
* Light & Dark Mode
* Dashboard Summary
* 3 Different Data Table Pages
* User Input Form Page  
* Calendar Integration
* FAQ Page
* 4 Different Charts

## In order to run 
* install node module with npm install
* start the application with npm start
